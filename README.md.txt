# Caesar Cipher Encryption and Decryption

Program ini adalah contoh implementasi sederhana dari metode enkripsi dan dekripsi Caesar Cipher menggunakan Python.

## Cara Kerja
- Program ini dapat digunakan untuk mengenkripsi dan mendekripsi pesan dengan metode Caesar Cipher.
- Pesan yang dienkripsi akan digeser karakternya sebanyak n posisi, dan pesan yang didekripsi akan digeser ke arah sebaliknya.
- Program akan mempertahankan karakter non-alfabetik seperti spasi dan tanda baca.

## Cara Penggunaan
1. Jalankan program dengan Python.
2. Masukkan pesan yang akan dienkripsi
3. Masukkan jumlah pergeseran (shift) yang diinginkan.
4. Program akan menghasilkan pesan terenkripsi atau terdekripsi.

## Contoh Penggunaan
### Hasil
Pesan Asli: halo semuanya aku lagi makan
Pesan Terenkripsi: kdor vhpxdqbd dnx odjl pdndq
Pesan Terdekripsi: halo semuanya aku lagi makan
